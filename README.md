# 🧮 Java Calculator

A simple yet powerful **Java-based calculator** that performs basic arithmetic operations along with modulus and decimal calculations.
This project is designed for **beginners** to understand user input, operators, and logical flow in Java.

---

## ✨ Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Modulus (%)
* Supports **decimal numbers (floating point values)**

---


## 🚀 How to Run

1. Clone the repository:

   ```
   git clone https://github.com/Kartik208-web/crc_java.git
   ```

2. Open in any Java IDE (VS Code / IntelliJ / Eclipse)

3. Compile and run:

   ```
   javac Calculator.java
   java Calculator
   ```
